# 🤖 Multi-Agent Research System

An AI-powered multi-agent system that performs automated research using:

- 🔍 Tavily Search Tool (real-time web search)
- 🌐 BeautifulSoup (web scraping)
- 🤖 OpenAI LLM (reasoning + analysis)
- 🔗 LCEL (Runnable pipelines)
- 🧠 ReAct Agents

---

## 🚀 Features

- Multi-agent architecture (Search Agent + Reader Agent)
- Tool-based reasoning
- Dynamic web scraping + summarization
- LCEL pipeline orchestration
- Scalable design for research automation

---

## 🛠️ Tech Stack

- Python
- LangChain / LangGraph
- OpenAI API
- Tavily API
- BeautifulSoup
